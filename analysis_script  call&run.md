# Running Analysis Notebook

## Purpose
Runs regression, correlation, hypothesis testing, and visualization using `analysis_utils.py`.

## Workflow
1. Load dataset (`stores.csv`).
2. Run regression (`prepare_regression`).
3. Check correlation (`check_correlation`).
4. Perform hypothesis test (`compare_groups`).
5. Generate charts (`plot_sales_by_group`).

## Employer Takeaway
Demonstrates statistical analysis and visualization skills, with reusable utility functions.
